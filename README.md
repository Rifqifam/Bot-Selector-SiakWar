# Bot-Selector-SiakWar
Supaya Cepet buat select matakuliah saat siakwar, page load langsung ke centang smua matkul yang di pilih

#### 1. Install Chrome Extension [TamperMonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo/related?hl=id)
#### 2. Teken tombol "Create New Script" di extension tampermonkey
![image](https://github.com/Rifqifam/Bot-Selector-SiakWar/assets/88616834/b89bb003-83b1-4d75-883a-624408d1d9cd)
#### 3. Masukin Kode [INI](https://github.com/Rifqifam/Bot-Selector-SiakWar/blob/main/siakwarbot.js) di create new script tampermonkey tadi
#### 4. Masukin Nama mata kuliah kalian di array 'matkul_code'
![image](https://github.com/Rifqifam/Bot-Selector-SiakWar/assets/88616834/0e0a421e-9e6b-4bee-b7a2-21385d324bfa)
#### 5. Contoh nama mata kuliah adalah, seperti tulisan dibawah yang berwarna hijau
![image](https://github.com/Rifqifam/Bot-Selector-SiakWar/assets/88616834/c03b634f-54bb-4872-acce-6d92906912f8)
#### 6. Jangan lupa untuk enable script yg sudah disave dengan mengakses dashboard pada extension tampermonkey
#### 7. Refresh halaman 'isi/ubah IRS' atau dengan menekan tombol F5
